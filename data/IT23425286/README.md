## Project Overview
This project contains automated functional and UI test cases implemented using Playwright for the Singlish to Sinhala Translator web application.

## Prerequisites
- Node.js (version 18 or later)
- Git

## Installation Steps
1. Clone the repository:
   git clone <REPO_LINK>

2. Navigate to the project directory:
   cd IT23425286

3. Install dependencies:
   npm install

4. Install Playwright browsers:
   npx playwright install

## Running Tests
To run all tests in Chromium:
npx playwright test --project=chromium

To run tests in headed mode:
npx playwright test --project=chromium --headed

## Test Case Documentation
Manual test cases are documented in the Excel file:
test-cases/IT23425286.xlsx

These test cases were used as a reference to design the automated Playwright scripts.
